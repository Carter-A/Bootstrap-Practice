Languages

HTML5

The HTML code I write is semantic, SEO friendly, and most importatly, beautiful. I primarily use Bootstrap as a frame work to produce responsive, mobile first sites. It is now more important than ever to ensure content can be accessed on huge range of devices.

CSS

What an ugly web we would have without css. My css is well organized and properly annotated. On larger projects multiple css files, paired with a indexing system ensures whom ever is working on it wastes no time, and is never lost.

JavaScript

The language that brings life to the web. I favor jQuery for speed of development and seemingly endless applicability. I have the ability to code in basic JavaScript, however, find it more efficient to use jQuery. I am currently learning AngularJs to further challenge my skill set and build full web applications.

Tools

Just a few of my favorites

Sublime Text

My text editor of choice currently is Sublime Text 3. It is obviously the industry standard and for good reason. When combined with Emmet it makes my life so much easier. I have been experimenting with Brackets by Adobe. It has some great features (live preview)

Sublime Text

GitHub or Git

I use GitHub as my version control system. Operating from the command line, projects I am working on are held centrally at GitHub and can be accesssed by anyone. A powerful tool and community no web dev should be without.

GitHub

Yeoman

Yeoman is a web scaffolding tool that allows me to jumpstart any project. It provides the starting architecture for any website. Because it comes prepackaged with Bootstrap it is an easy one stop to start a project. Also easily installed using Node.js.
